## Getting Started

### Installation locally

1. Server Side
  ```sh
    git clone https://github.com/tobiasz18/RestApi_with_React
    npm install 
    npm start
  ```

2. Client side
```sh
  git clone https:://github.com/your_username_/Project-Name.git || cd client

  then go to client folder and type in terminal >
    npm install 
    npm run build
    npm start
   
```
